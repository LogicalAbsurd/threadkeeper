### Architecture decisions encoded in repo's CLAUDE.md
- All scrapers implement `listConversations()`, `loadConversation(id)`, `parseMessages()` triad
- CONTENT_READY handshake with chatId matching for SPA navigation safety
- Three-technique lazy-load loop: scrollTop=0, overscroll, scrollIntoView({block: 'end'}) — proved necessary for Gemini, doesn't apply to ChatGPT (API approach)
- Background owns navigation via tabs.update(); content scripts never navigate
- bulk export accumulates only when outputMode is 'combined' or 'both'
